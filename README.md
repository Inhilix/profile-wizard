Profile Wizard

A multi-step React application that builds and scores a student university application profile.

## Features
- 4-step wizard: Academic Background, Test Scores, Goals, Review
- Live score computed from GPA, IELTS, and profile completeness
- Animated SVG score ring with colour-coded bands
- localStorage persistence — progress survives page refresh
- Reusable components: Button, ScoreCard, StepIndicator

## Tech Stack
- React 19, Vite, JavaScript (JSX), CSS custom properties

## Run Locally
npm install
npm run dev
